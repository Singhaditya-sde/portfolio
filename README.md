# My Portfolio Website 🚀

A modern developer portfolio built with **React, Vite, TypeScript, and Tailwind CSS**.
This project showcases my projects, skills, and experience with a clean UI and optimized developer workflow.

---

## ✨ Features

* ⚡ Fast development with **Vite**
* ⚛️ Built using **React 19**
* 🧠 Type safety with **TypeScript**
* 🎨 Styling with **Tailwind CSS**
* 🧩 UI utilities using **Radix UI** and **shadcn components**
* 🧹 Code formatting with **Prettier**
* 🔍 Code quality enforced with **ESLint**
* 🪝 **Husky + lint-staged** for pre-commit checks
* 📦 Optimized production build

## 🚀 Getting Started

### 1️⃣ Clone the repository

```
git clone https://github.com/yourusername/my-portfolio.git
```

### 2️⃣ Navigate into the project

```
cd my-portfolio
```

### 3️⃣ Install dependencies (using Bun)

```
bun install
```

### 4️⃣ Start the development server

```
bun run dev
```

The app will start at:

```
http://localhost:5173
```

---

## 📜 Available Scripts

| Script             | Description                   |
| ------------------ | ----------------------------- |
| `bun run dev`      | Start development server      |
| `bun run build`    | Create production build       |
| `bun run preview`  | Preview production build      |
| `bun run lint`     | Run ESLint checks             |
| `bun run lint:fix` | Fix lint issues automatically |
| `bun run format`   | Format code using Prettier    |

---

## 🧹 Code Quality

This project uses **ESLint + Prettier** for maintaining clean and consistent code.

Before every commit:

* ESLint checks the code
* Prettier formats files automatically

This is handled by **Husky + lint-staged**.
